# 🎯 초등학교 3학년 사회과 지역화 수업자료 생성기 - 단계별 개선 계획

## 프로젝트 현황

**기술 스택**: Next.js 14, React, TypeScript, Tailwind CSS

**현재 상태**: UI 프로토타입 완성, 더미 데이터로 동작

**주요 페이지**:
- 홈, 워크스페이스(통합 페이지)
- 지역 선택, PPT 생성, 다운로드
- 관리자: 템플릿 관리, 통계
